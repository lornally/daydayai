###### 推荐设置

| 设置                                    | 推荐值                     | 理由                              |
| --------------------------------------- | -------------------------- | --------------------------------- |
| **Rayout Rate** (配牌率)                | 96%                        | 配牌成功概率提升                  |
| **Odds Rate** (赔率/倍率)               | 没试出来对赢率的影响       | 随便投币, 无所谓                  |
| **Last Chance** (最后机会)              | on                         | 给海底捞月                        |
| **Auto Reach** (自动听牌)               | on                         | 听牌就自动打                      |
| `Yakuman Bonus Cycle`(役满奖励周期)     | `Every 300 coins`          | 奖励更常见                        |
| `Yakuman Bonuses Per Cycle`(每周期奖励) | `2`                        | 每周期奖励更多                    |
| `Game Title`(游戏名)                    | `Mahjong Tenkaigen Part 2` | 这样可以打开 `Show Yakuman Table` |
| **Show Yakuman Table** (显示役满表)     | ON                         | 方便看役满牌型                    |

###### 推荐默认

| 设置                                                    | 默认值   | 说明             |
| ------------------------------------------------------- | -------- | ---------------- |
| `Computer Strength`(电脑强度)                           | `Normal` | 没有 `Weak` 选项 |
| **Renchan Rate** (连庄率)                               | on       |                  |
| **Tenkaigen Day** (天开眼日)                            | normal   |                  |
| **SP Express Frequency** (SP 表现频率) sp=特殊表现=动画 | normal   |                  |
| **Don Den Count** "Don Den" 赌博模式的可用次数          | 8        |                  |

**破解版 vs 原版区别：**

- 破解版通常移除了电脑作弊、听牌必胡等
- 破解版可能去除了某些限制
- 原版体验更接近街机
