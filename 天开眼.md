## 电子基盘 (Mahjong Electron Base) 版本区别

| 版本         | 文件名         | 说明                            |
| ------------ | -------------- | ------------------------------- |
| **mjelctrn** | mjelctrn.zip   | 原版 (parts 2 & 4, Japan)       |
| **mjelct3**  | mjelct3.zip    | 原版 (parts 2 & 3, set 1) ⭐推荐 |
| mjelct3a     | mjelct3a.zip   | 修改版 (set 2)                  |
| mjelct3b     | mjelct3b.zip   | 修改版 (set 3)                  |
| mjelct3bl    | mjelct3bl.zip  | 破解版 (bootleg set 1)          |
| mjelct3bla   | mjelct3bla.zip | 破解版 (bootleg set 2)          |
| mjelct3blb   | mjelct3blb.zip | 破解版 (bootleg set 3)          |
| mjelctrb     | mjelctrb.zip   | 破解版 (bootleg)                |

**破解版 vs 原版区别：**
- 破解版通常移除了电脑作弊、听牌必胡等
- 破解版可能去除了某些限制
- 原版体验更接近街机
