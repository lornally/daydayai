# 天开眼 (Tenkai) DIP 设置详解

## 🎮 DIP Switches 解释

DIP Switches 是街机主板的物理开关，用于调整游戏设置。在 MAME 中通过 Tab 菜单模拟。

---

## 📋 各设置项含义

### 1. **Rayout Rate** (配牌率)
| 设置 | 含义 |
|------|------|
| 高 | 更容易配到好牌（容易胡） |
| 低 | 配牌更随机（正常） |
| **推荐** | 普通或低 |

### 2. **Odds Rate** (赔率/倍率)
| 设置 | 含义 |
|------|------|
| 高 | 胡牌得分更高 |
| 低 | 胡牌得分正常 |
| **推荐** | 高（想多赢分） |

### 3. **Yakuman Bonus Cycle** (役满奖励周期)
| 设置 | 含义 |
|------|------|
| 短 | 更容易出役满牌型 |
| 长 | 役满更难出现 |
| **推荐** | 短（想胡大牌） |

### 4. **Payout Mode** (出分模式)

| 设置        | 含义                         | 推荐               |
| :---------- | :--------------------------- | :----------------- |
| **Key-Out** | 钥匙出分，需手动操作才能退币 | ❌ 街机厅用，麻烦   |
| **Hopper**  | 自动退币，赢分自动到账       | ✅ **推荐**，最方便 |

### 5. **Hopper Polarity** 
- inverted

### 6. **Tenkaigen Day** (天开眼日)
- 特殊日子设置
- 可能影响当天运势或特殊事件

### 7. **Renchan Rate** (连庄率)
| 设置 | 含义 |
|------|------|
| 高 | 庄家更容易连庄 |
| 低 | 连庄较难 |
| **推荐** | 低（公平点） |

### 8. **Auto Reach** (自动听牌)
| 设置 | 含义 |
|------|------|
| ON | 系统自动帮你宣告听牌 |
| OFF | 手动宣告听牌 |
| **推荐** | OFF（手动更有乐趣） |

### 9. **Last Chance** (最后机会)
| 设置 | 含义 |
|------|------|
| ON | 最后一张牌时给提示 |
| OFF | 无提示 |
| **推荐** | ON |

### 10. **Don Den Button** ( don den 按钮)
- 特殊赌博按钮设置
- 控制" don den "功能是否可用

### 11. **Gal Select** (女孩选择)
- 选择对手女孩类型
- 可能包含不同风格/难度

### 12. **Gal Pose** (女孩姿势)
- 赢牌后女孩的奖励画面姿势设置
- 可能包含不同尺度/类型

### 13. **SP Express Frequency** (SP 表现频率)
- 特殊表现/动画的出现频率
- 高 = 更多特殊效果

### 14. **Computer Strength** (电脑强度)
| 设置 | 含义 |
|------|------|
| Weak | 电脑弱（容易赢） |
| Normal | 正常 |
| Strong | 电脑强（难赢） |
| **推荐** | Normal 或 Weak |

### 15. **Service Count** (服务次数)
- 投币后免费游玩的次数
- 通常设为 0（正常投币）

### 16. **Don Den Count** (Don Den 次数)
- "Don Den" 赌博模式的可用次数
- 控制赌双倍的机会

### 17. **Show Yakuman Table** (显示役满表)
| 设置 | 含义 |
|------|------|
| ON | 显示役满牌型列表 |
| OFF | 不显示 |
| **推荐** | ON（方便查看役满） |

---

## 🎯 推荐配置（简单模式）

如果想轻松玩：

| 设置 | 推荐值 | 理由 |
|------|--------|------|
| Rayout Rate | 高 | 容易配好牌 |
| Odds Rate | 高 | 赢分多 |
| Computer Strength | Weak | 电脑弱容易赢 |
| Yakuman Bonus Cycle | 短 | 容易出役满 |
| Auto Reach | OFF | 手动更有控制感 |
| Show Yakuman Table | ON | 方便看役满牌型 |

---

## 💡 如何修改 DIP 设置

```
1. 游戏中按 Tab
2. 选择 "DIP Switches"
3. 使用方向键移动
4. 按左/右键切换设置
5. 按 Esc 退出（自动保存）
```

---

## ⚠️ 注意

- DIP 设置会影响游戏平衡
- 设置过高可能失去挑战性
- 街机厅通常设为中等到困难
- 个人玩可以设为简单享受游戏
