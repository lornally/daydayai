下面给你一个“日麻（立直麻将）一页版”，尽量简洁，但把关键坑点和常见番都带上。默认按现在最常见的标准立直规则讲，个别平台会有少量变体。

**一、核心规则**
1. 4人打，目标是胡牌拿点。
2. 通常胡牌形：`4组面子 + 1个雀头`
   面子 = 顺子 `123` / 刻子 `777` / 杠子 `9999`
3. 特殊胡牌形还包括：
   - `七对子`
   - `国士无双`
4. 胡牌方式：
   - `荣和`：别人打出你要的牌，你胡
   - `自摸`：自己摸到
5. 最关键：`必须有役（至少1番）才能胡`
   - 只有宝牌/红宝牌，`没有役`，不能胡
6. 副露规则：
   - `吃`：只能吃上家打的牌
   - `碰`：谁打都能碰
   - `杠`：明杠、暗杠、加杠
7. `门前清` = 没有吃碰明杠；很多役要求门清。
8. `立直`：
   - 只能门前
   - 已经听牌
   - 打出1000点立直棒
   - 立直后手牌基本不能改，主要就是摸切

**二、你提到的关键点：振听（非常重要）**
振听 = `你不能荣和，只能自摸`

有3种常见振听：

1. `舍牌振听`
   - 如果你自己打过的牌里，有任何一张是你当前听牌能胡的牌，你就振听。
   - 例：你听 `3万/6万`，而你以前自己打过 `3万`，那别人打 `3万` 或 `6万` 你都不能荣和，只能自摸。
   - 如果你还没立直，之后改听口，且新听牌不含自己打过的那张，振听可解除。

2. `同巡振听`
   - 这一巡里，有人打出你能胡的牌，但你没胡；那么在你下次摸牌前，别人再打任何你能胡的牌，你也不能荣和。
   - 这就是你说的“对面之前打过一次，第二次再打出来，我好像不能胡”的来源。
   - 更准确地说：`如果你中间还没轮到自己摸牌，那第二次也不能胡。`
   - 如果你已经摸过牌了，而且没立直，那一般这个同巡振听就解除了。

3. `立直后漏胡振听`
   - 立直后，如果有人打出你能胡的牌你没胡，那通常这一局后面你都不能再荣和了，只能自摸。
   - 因为立直后不能改手，振听会一直持续。

一句话记：
- `自己打过能胡的牌 -> 长振听`
- `这巡别人打了你没胡 -> 到你下次摸牌前不能荣和`
- `立直后漏胡 -> 基本这局都只能自摸`

**三、计分你只要先记住这些**
1. 胡牌看 `番` 和 `符`
2. 新手先重点记 `番`；`符`稍复杂，但不用一开始全背
3. 常见打点线：
   - `满贯`：5番，或特殊高符3-4番
   - `跳满`：6-7番
   - `倍满`：8-10番
   - `三倍满`：11-12番
   - `役满`：役满牌型
4. `宝牌`、`里宝牌`、`杠宝牌`、`赤宝牌`都只是“加番”，`不算役`

**四、常见役（番种）总表**

**1番**
- `立直`：门前听牌后宣言
- `门前清自摸和`：门前自摸
- `一发`：立直后一巡内胡，期间没人吃碰杠
- `断幺九`：全手没有1、9、字牌
  - 现在多数规则允许副露后也算，叫“食断”
- `平和`：门前，4个顺子，雀头非役牌，听牌为两面听
- `一杯口`：门前，同一种花色里有两组相同顺子
- `役牌`：白、发、中；自风；场风
  - 每一种单独算1番
- `海底摸月`：最后一张自摸胡
- `河底捞鱼`：最后一张荣和胡
- `岭上开花`：杠后补牌自摸胡
- `抢杠`：别人加杠时抢胡

**2番**
- `七对子`：7个对子，门前限定
- `对对和`：全是刻子/杠子
- `三暗刻`：3组暗刻
- `三杠子`：3个杠
- `小三元`：白发中里，2组刻子 + 1组雀头
- `混老头`：全是1、9、字牌，且通常由刻子构成
- `三色同刻`：三种花色同数字刻子
- `三色同顺`：同一个顺子分别出现在万筒索
  - 门前2番，副露1番
- `一气通贯`：同一花色123+456+789
  - 门前2番，副露1番
- `混全带幺九`（混全/全带）：每组都带1、9或字牌
  - 门前2番，副露1番
- `双立直`：第一巡内立直，且中间没人鸣牌

**3番**
- `纯全带幺九`（纯全）：每组都带1、9，且没有字牌
  - 门前3番，副露2番
- `二杯口`：门前，两组一杯口
- `混一色`：一种花色 + 字牌
  - 门前3番，副露2番

**6番**
- `清一色`：全是一种花色，没有字牌
  - 门前6番，副露5番

**役满**
- `国士无双`：13种幺九牌体系
- `四暗刻`：4组暗刻
- `大三元`：白发中三组刻子
- `小四喜`：风牌3组刻子 + 1对风牌
- `大四喜`：4组风刻
- `字一色`：全字牌
- `清老头`：全是1、9牌
- `绿一色`：只用发和绿色索牌
- `九莲宝灯`：清一色特殊门前牌型
- `四杠子`：4个杠
- `天和`：庄家起手即胡
- `地和`：闲家第一摸即胡

**五、几个很容易搞错的点**
1. `有宝牌不等于有役`
   - 比如你有2张宝牌，但手里一个役都没有，还是不能胡。
2. `副露会让很多役降番或直接失效`
   - 比如立直、平和、一杯口、门前自摸都要求门前。
3. `平和`要求很严
   - 雀头不能是役牌
   - 通常要两面听
4. `断幺九`多数平台可副露，但不是所有古早规则都行。
5. `七对子`不是“二面子一对子一对子……”那种普通牌型，它是独立牌型。
6. `振听只影响荣和，不影响自摸`
